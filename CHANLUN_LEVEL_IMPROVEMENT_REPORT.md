## 三、用户问题的答案

### Q1: 三买是什么级别的三买？30分钟，1h，还是4h？

**完整答案**：

这是一个双维度问题，需要明确两个概念：

**维度1：时间级别**
- 当前系统使用**4小时K线**
- 时间级别 = 4h（中线级别）

**维度2：缠论级别**
- 基于1级中枢的三买
- 缠论级别 = 1级三买（短线信号）

**综合答案**：
```
这是4h时间级别下的1级三买
交易意义：中线级别的短线买点
建议持仓：2-7天
```

**解读**：
- 4h时间级别 = 中线级别
- 1级中枢 = 短线中枢
- 1级三买 = 短线买点
- **综合**：在中线级别下的短线买点，适合2-7天交易
