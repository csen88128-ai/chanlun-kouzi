# 缠论多智能体分析系统 v4.0

基于 LangGraph 和大语言模型的缠论全自动多智能体市场分析系统。

## v4.0 更新内容 ⭐

### 新增功能

#### 1. 研报自动生成 ⭐
- **Markdown格式研报**：结构化输出，包含完整分析内容
- **智能目录**：自动生成目录导航
- **多维度展示**：结构、动力学、情绪、跨市场、链上、决策全维度展示
- **风险提示**：自动生成风险等级和具体风险点
- **存储路径**：`reports/{symbol}_{interval}_{timestamp}.md`

#### 2. 可视化图表生成 ⭐
- **ASCII K线图**：字符画式K线图，支持在终端查看
- **图表描述**：数据概览、价格走势、波动性分析
- **价格范围显示**：自动计算最高最低价
- **时间轴标注**：清晰显示时间分布
- **存储路径**：`reports/{symbol}_{interval}_chart_{timestamp}.md`

#### 3. 历史决策回溯 ⭐
- **决策记录**：完整记录每次交易决策的所有参数
- **绩效统计**：自动计算胜率、盈亏比、平均置信度
- **方向分布**：做多/做空/观望的分布统计
- **历史报告**：生成Markdown格式的历史决策报告
- **数据持久化**：JSON格式存储，方便后续分析

#### 4. 研报智能体 ⭐
- **整合所有分析**：汇总结构、动力学、情绪、跨市场、链上分析
- **生成完整研报**：自动调用研报生成模块
- **历史决策整合**：包含历史决策统计信息
- **输出标准化**：统一的研报格式和内容

### v3.0 - 辅助维度分析

#### 1. 市场情绪分析

#### 恐慌贪婪指数
- **指数范围**: 0-100
- **极度恐惧** (0-24): 市场恐慌，可能接近底部
- **恐惧** (25-44): 市场情绪悲观，可能存在机会
- **中性** (45-55): 市场情绪平稳
- **贪婪** (56-75): 市场情绪乐观，可能存在泡沫
- **极度贪婪** (76-100): 市场过热，可能接近顶部

#### 资金费率
- **正费率**: 多头占优，多头支付给空头
- **负费率**: 空头占优，空头支付给多头
- **零费率**: 多空平衡

#### 爆仓数据
- 多单爆仓数量 vs 空单爆仓数量

## 决策优先级

1. **系统监控告警** - 最高优先级，任何告警暂停决策
2. **大级别结构** - 以大级别判断为准（权重30%）
3. **多维度共振** - 共振越多，置信度越高
4. **动力学确认** - 有背驰确认的信号优先（权重25%）
5. **市场情绪** - 极端情绪必须重视（权重15%）
6. **跨市场联动** - 宏观环境影响（权重15%）
7. **链上信号** - 基本面支撑（权重10%）
8. **模拟盘验证** - 胜率 < 45% 的信号降权（权重5%）
9. **风控否决** - 风控拥有一票否决权

## 缠论核心原则

系统严格遵循缠论理论体系：

1. **不臆测** - 结构未完成就说「未完成」，不强行定论
2. **不迎合** - 分析结论不因「希望市场涨/跌」而偏移
3. **不糊弄** - 有争议的结构标注「观察中」，不假装确定
4. **不甩锅** - 决策失误主动承认，不找借口
5. **不遗漏** - 关键风险必须上报，不得擅自忽略
6. **不造假** - 模拟盘结果如实记录，不美化
7. **不干扰** - 收到冷却信号后，立即停止生成信号
